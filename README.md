# Web Crawler mimi by Ismael Nascimento

## Requirements

For development, you will only need Node.js installed on your environement.

## Start local project

    $ cd lambda_list_files_page/
    $ node index
    --------------------------------
    $ cd lambda_list_pages/
    $ node index
